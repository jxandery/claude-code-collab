# 1.0.0 - Initial Release (2025-12-12)

## Added
- Real-time collaboration for Claude Code sessions via tmux
- User attribution with automatic username prefixing
- tmate and SSH-based collaboration modes
- join-claude-session.sh for simple input mode
- join-claude-session-split.sh for split-pane mode
- start-collaboration.sh interactive wizard for easy setup
- list-sessions.sh to view active tmux sessions
- download-from-server.sh for retrieving collaboration files
- Installation script with PATH configuration
- Documentation: collaborator/host instructions, server setup, troubleshooting
- Semantic versioning with bump-version.sh
